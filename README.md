# DPDK 高性能防火墙 （DFW）
基于 DPDK 实现的高性能用户态防火墙
# 主要功能
## 1. 高性能数据平面
- 基于 DPDK 的多队列、多 lcore 转发模型
- 批量收包、批量处理、批量发送
- 全用户态运行，绕过内核协议栈
## 2. ARP 子系统（L2）
- 维护 ARP 表（IP → MAC → 出口端口）
- 识别并响应发往本机 IP 的 ARP Request
- 在转发前自动解析下一跳 MAC
- 支持 ARP 表项支持老化与定期扫描
- 可配置生存时间与回收策略
## 3. 路由模块（L3）
- 支持静态路由表
- 基于目的 IP 做最长前缀匹配（LPM）查找
- 决定下一跳 IP 与出口端口
## 4. NAT 模块（L3/L4）
- 支持SNAT（源地址转换）
- 支持 DNAT（目的地址转换）
- 支持 NAT 表项支持老化与定期扫描
- 可配置生存时间与回收策略

## 5. ACL 模块（访问控制）
- 基于五元组的包过滤
- 支持 permit / deny 规则
